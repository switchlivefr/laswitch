Suppression de vos données — La SWiTCH

Dernière mise à jour : avril 2026

L'application La SWiTCH ne stocke aucune donnée personnelle sur ses serveurs. Votre identifiant Facebook est utilisé uniquement de façon temporaire lors de votre session pour identifier vos vidéos.

Si vous souhaitez supprimer l'accès de l'application à votre compte Facebook, rendez-vous dans les paramètres de votre compte Facebook :

Paramètres → Sécurité → Applications et sites web → La SWiTCH → Supprimer

Cette action supprime immédiatement tout accès de l'application à votre profil Facebook.
